Detailed Description

Initializes terms for v2-f simulation and provides some basic routines.

Author:
Gopal Yalla

This file defines the methods to set up the v2-f simulation, i.e., to get inputs, solve for the initial conditions, and defines struct of model constants, as well as defines basic functions such as log level outputs and writing results to a file.
