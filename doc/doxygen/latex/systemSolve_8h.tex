\subsection{Detailed Description}
Sets up system for gsl's multiroot solvers. \begin{DoxyAuthor}{Author}
Gopal Yalla
\end{DoxyAuthor}
This file defines the methods to set up the system $F(\xi)$ of finite difference aproximations as defined in the project proposal.
